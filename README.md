tinystack
=========

Little communication stack for embedded systems.

Aims:
- UDP style:
-- Fire & forget packets
-- Packet arrival not guaranteed
- No dynamic memory use
- Small packet overhead
- Robust error checking


Dev requirements
================
- python 2.x
- doit: http://pydoit.org/
